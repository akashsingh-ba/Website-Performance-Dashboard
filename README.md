# Website-Performance-Dashboard
Website Performance Dashboard
Project Overview

This Power BI project analyzes website performance metrics to understand visitor behavior, traffic effectiveness, and conversion trends. The dashboard provides interactive insights into bounce rates, conversion rates, page views, session duration, visitor types, and geographic performance.

**Tools Used**
1.Power BI
2.Power Query
3.Data Visualization
4.KPI Cards
5.Slicers
6.Maps
7.DAX

**Dataset Features**
1.Visitor ID
2.Visitor Type
3.Traffic Source
4.Location
5.Page Views
6.Session Duration
7.Bounce Rate
8.Conversion Rate
9.Exit Pages

**Key Metrics**
1.Average Bounce Rate
2.Average Conversion Rate
3.Average Page Views
4.Average Session Duration

**Business Questions Answered**
1.What is the average bounce rate by visitor type?
2.What is the average conversion rate by visitor type?
3.Which traffic source generates the highest conversion rate?
4.How does conversion rate vary by location?
5.Who are the top 100 visitors based on conversion rate?
6.How do exit pages impact website performance?

**Dashboard Features**
1.Interactive Exit Page Filter
2.KPI Cards
3.Donut Charts
4.Bar Chart
5.Map Visualization
6.Top Visitors Analysis Table

**Key Insights**
1.Conversion rates vary across traffic sources.
2.Visitor type influences bounce and conversion behavior.
3.Geographic regions show differences in conversion performance.
4.Exit page analysis helps identify potential drop-off points.
5.High-performing visitors can be identified through conversion metrics.
