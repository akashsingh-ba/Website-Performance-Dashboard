# Website Performance Dashboard

## Project Overview

This Power BI project analyzes website performance data to understand visitor behavior, traffic source effectiveness, and conversion trends. The dashboard provides interactive insights into key website metrics, helping identify opportunities to improve user engagement and conversions.

## Tools Used

- Power BI
- Power Query
- DAX
- Data Visualization
- KPI Cards
- Slicers
- Maps

## Dataset Information

The dataset contains website visitor information including:

- Visitor ID
- Visitor Type
- Traffic Source
- Location
- Page Views
- Session Duration
- Bounce Rate
- Conversion Rate
- Exit Pages

## Key Performance Indicators (KPIs)

- Average Bounce Rate
- Average Conversion Rate
- Average Page Views
- Average Session Duration

## Business Questions Answered

1. What is the average bounce rate by visitor type?
2. What is the average conversion rate by visitor type?
3. Which traffic source generates the highest conversion rate?
4. How does conversion rate vary across different locations?
5. Who are the top 100 visitors based on conversion rate?
6. How do exit pages impact website performance?

## Dashboard Features

- Interactive Exit Page Filter
- KPI Cards
- Donut Charts
- Bar Chart
- Map Visualization
- Top 100 Visitors Analysis

## Key Insights

- Visitor type influences both bounce rate and conversion rate.
- Traffic sources contribute differently to website conversions.
- Geographic regions show variations in conversion performance.
- Exit page analysis helps identify potential user drop-off points.
- High-converting visitors can be identified through conversion rate analysis.
